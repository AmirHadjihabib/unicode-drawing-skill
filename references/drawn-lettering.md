# Drawn Lettering

Words as visual expression. Two territories: typographic styles for shaping letters, and expressive treatments for shaping meaning.

## Typographic Styles

Different ways to construct letterforms from marks. Each style gives words a different voice.

```

## Words Built from Marks

The substance surrounding or constructing a word changes what it feels like. Some approaches:

### Words Carried by Marks

The mark wraps the word — the texture becomes the voice.

```
  ░ quiet ░
  ▒ steady ▒
  ▓ STRONG ▓
  █ CERTAIN █
```

```
  · · dream · ·
  ░░░ dream ░░░
  ▓▓▓ DREAM ▓▓▓
  ███ DREAM ███
```

```
  ≈ ≈ calm ≈ ≈
  ✦ ✦ wonder ✦ ✦
  ◎ ◎ warmth ◎ ◎
```

### Block Letters

Clean, solid `█` reads clearly at any size.

```
HI:
█ █  █
█ █  █
███  █
█ █  █
█ █  █
```

```
DREAM:
██▄  ███  █▀▀  ▄▀▄  █▄█
█ █  █▄▀  █▀   █▀█  █ █
█▀   █ █  ▀▀▀  ▀ ▀  ▀ ▀
```

### Dot Letters

`·` gives airiness — letters breathe.

```
GO:
·····  ·····
·      ·   ·
· ···  ·   ·
·   ·  ·   ·
·····  ·····
```

### Braille Letters

The density of `⣿` makes thick, rich letterforms.

```

  ⣿⡇   ⣿⡇ ⣿⡇  ⣿⡇⣿⡇

⣿⣿⡇   ⣿⡇ ⣿⡇    ⣿⡇
 ⣿⡇    ⣿⣿⣿⡇    ⣿⡇

## Mixing Styles and Treatments

The most expressive drawings combine typographic style with spatial treatment:

```
  · go ·

     ░ go ░

        ▒ GO ▒

           ▓ GO ▓

  ████████████████████
  ██      G O      ██
  ████████████████████
```

```
  ✦   ·   ✦
    D R E A M
  ·   ✦   ·   ✦
          in
        ░ dots ░
      and
    ▓▓ BLOCKS ▓▓
  ✦   ·   ✦
```

The drawing of the word changes what the word means. Trust what feels right.
